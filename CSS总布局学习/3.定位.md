## 相对定位

1. 语法：position：relative
  - 相对于原理啊的位置，进行指定得偏移，相对定位得话，他仍然在标准文档流中，原来的位置不变
  
```css
        #first{
            position: relative;  
            top:-20px;
            left: 10px;
            right：20px；
        }
```
        
## 绝对定位
  ### 基于xxx进行定位，进行上下左右
      1. 如果没有存在父元素的定位的前提下，他会相对于浏览器进行定位
      2. 假设父元素存在定位，通常会相对于父元素进行偏移
      3. 在父元素范围内进行移动
      - 相对于父级或者浏览器的位置，进行指定偏移的时候，绝对定位的话，它不存在标准流文档中，原本的位置不会会保留
